In this workshop you will learn the fundamentals of the
[Air](https://github.com/feldroy/air) web framework for Python. Air is built
on top of FastAPI, Starlette, and Pydantic, and introduces **Air Tags** — Python
classes that render directly to HTML — as a streamlined way to build web pages
without leaving Python.

Air is created by the authors of *Two Scoops of Django* and is designed to let
you serve both web pages and REST APIs from a single application.

## What You Will Learn

By the end of this workshop you will be able to:

- Examine and run an Air web application
- Build HTML pages using Air Tags
- Define routes with `@app.get()` and the `@app.page` decorator
- Handle path parameters to create dynamic routes

## Getting Started

A starter project has been set up for you in the `~/exercises` directory with
Air already installed. Both terminal sessions are ready to use — no additional
setup is required.
